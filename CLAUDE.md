# 纯文本差异对比工具 - 项目开发计划

> **项目名称**：文本差异对比工具
> **技术栈**：Vue3 + Vite + TypeScript + diff-match-patch + UnoCSS + Element Plus
> **开发阶段**：从0到1完整搭建
> **创建日期**：2025-11-22

## 📋 项目概述

这是一个基于Web的纯文本差异对比工具，专注于**精细化文本差异分析**，使用diff-match-patch算法实现字符级别的精确差异检测。

**核心价值**：
- 字符级精确差异检测（非行级）
- 可视化差异展示（红绿蓝三色编码）
- HTML导出功能
- 跨平台兼容性

### 🔧 开发服务器使用原则
1. **按需启动**: 仅在需要时启动开发服务器 (`pnpm run dev`)
2. **及时清理**: 完成功能验证后立即关闭所有开发服务器进程
3. **端口管理**: 避免长时间占用端口资源（3000、3001、3002等）

### 🧹 服务清理流程
1. **关闭后台进程**: 确保所有 `pnpm run dev` 后台进程已终止
2. **进程检查**: 使用以下命令确认没有残留进程
   ```bash
   ps aux | grep "pnpm run dev\|vite\|node.*dev" | grep -v grep
   ```
3. **强制清理**: 如有残留进程，使用 `kill -9 <pid>` 强制关闭
4. **资源释放**: 确认所有端口已释放，可正常重新使用

### ⚠️ 注意事项
- **避免并行运行**: 不要同时运行多个开发服务器实例
- **子进程清理**: 确保Vite相关的子进程也完全终止
- **端口冲突**: 定期检查端口占用情况
- **开发习惯**: 每次开发完成后执行清理流程
